# Installing


Clone the flake

```sh
git clone https://github.com/raphaelladinig/nixos
```
```sh
cd nixos
```

Enter nix-shell

```sh
nix-shell
```

Install flake

```sh
just install inspiron
```

Move the flake to /mnt/persist/etc/nixos

Boot in to the installation

Change the passwords of the users & deploy secrets
